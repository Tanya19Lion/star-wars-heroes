# Star Wars Heroes Project

This is a React + Vite project for displaying Star Wars heroes with details, lists, and interactive cards. The project uses **React Testing Library** and **Vitest** for testing.

## Setup

1. Clone the repository:

```bash
git clone https://github.com/Tanya19Lion/star-wars-heroes.git
cd star-wars-heroes

# Install dependencies
npm install

# Run project
npm run dev

# Run all tests
npx vitest run

# Run in watch mode (recommended during development)
npx vitest
